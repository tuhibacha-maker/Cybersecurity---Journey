 Day 3: Basic Networking Commands 

## 🌐 Ping Command
ping google.com  
👉 Checks if a server or internet connection is reachable  

---

## 🖥️ IP Address Check
ip a  
👉 Displays your IP address and network details  

---

## 🔍 Domain to IP
nslookup google.com  
👉 Finds the IP address of a domain  

---

## 🛣️ Route Check
traceroute google.com  
👉 Shows the path taken by data to reach the destination  

---

## 🔗 Open Ports
netstat -tuln  
👉 Displays open ports and active connections  

---

## ✨ Conclusion
These basic commands are essential for understanding networking and troubleshooting
